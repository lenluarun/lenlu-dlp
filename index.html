Initialization Sequence

Deploy the local server engine in seconds.

~/lenlu-dlp/setup.sh

# 1. Clone the repository
$git clone https://github.com/lenluarun/lenlu-dlp.git
$cd lenlu-dlp

# 2. Install Python dependencies
$pip install fastapi uvicorn yt-dlp

# 3. Launch the Application Server
$python run_gui.py
                    

# Fast launch on Windows using the Python Launcher
C:\>py run_gui.py
                    

# Ensure pip3 and python3 are used on Unix systems
$pip3 install fastapi uvicorn yt-dlp
$python3 run_gui.py

Frontend Deployment Hook

Hosting the static index.html on Vercel or GitHub Pages? You can route the local python backend directly to your live URL.

Inside the Web UI System Settings, set the DEPLOYED_FRONTEND_URL to your live address. Your browser will automatically establish a secure link to 127.0.0.1:8000.

CLI Console Commands

Master the built-in custom terminal emulator.

> status

Retrieve current connection mode (Local/Serverless) and active background process details.

> clear

Purge the standard output feed buffer in the console tab to remove old logs.

> library

Instantly switch views and refresh the local downloads media catalog.

> select-folder

Triggers the native OS file picker to re-route your media download directory.

> site-folder

Reverts the target downloads directory to the default project installation fallback path.

> help formats

Displays an interactive guide on utilizing the advanced stream Matrix filters.
