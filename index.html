Dual-engine mode

Local or serverless —
you choose.

Run the FastAPI local server for full yt-dlp power with custom flags, subtitles, speed limiting, and audio extraction. Or flip to serverless Cobalt API for zero-install browser downloads.

Local FastAPI Cobalt Serverless
mode-selector.js 
const mode = 'local' // or 'serverless'
 
if (mode === 'local') {
  connectLocalServer({
    host: '127.0.0.1',
    port: 8000,
    engine: 'yt-dlp'
  });
} else {
  connectCobalt({
    endpoint: 'cobalt.tools/api',
    streaming: true
  });
}

System design

Hybrid deployment model.

One frontend, two backend modes. The UI auto-detects your environment and connects to the right engine.

Mode A — Local Engine

FastAPI + yt-dlp

Full control. Run the Python server on your machine for advanced flags, format selection, subtitles, audio extraction, and speed limits.

1
Browser FrontendSends request to 127.0.0.1:8000
2
FastAPI ServerValidates request, calls yt-dlp
3
yt-dlp CoreExtracts stream, applies flags
4
Local StorageFile saved to your download folder
Mode B — Serverless

Cobalt API

Zero setup. Uses the public Cobalt API node — no Python, no server, no install. Great for quick grabs on any device.

1
Browser FrontendSends URL to Cobalt API node
2
Cobalt NodeExtracts direct media stream URL
3
RedirectReturns download link to browser
4
Browser DownloadDirect stream via browser

Frontend Deployment Hook

Hosting the static index.html on Vercel or GitHub Pages? Set DEPLOYED_FRONTEND_URL inside the Web UI system settings to your live address — your browser auto-connects to 127.0.0.1:8000.

CLI reference

Console commands.

Master the built-in terminal emulator with these commands.

Command Description
> status Show current connection mode (Local/Serverless) and active process details.
> clear Purge the output feed buffer in the console tab.
> library Switch to and refresh the local downloads media catalog.
> select-folder Open the native OS file picker to re-route your download directory.
> site-folder Revert download directory to the default project installation path.
> fetch Fetch video metadata and available streams for the current URL.
> download Start the quick download using the selected preset format.
> settings Open the settings tab directly from the console.
> matrix Open the format matrix tab to inspect all available streams.
> help formats Display the interactive stream format guide and matrix filter tips.
> help download Show quick-download usage tips and preset format guide.
> --help Display the full command list with descriptions.
