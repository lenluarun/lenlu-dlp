Engine

Powered by yt-dlp &
Cobalt API.

The backend is a slim Python FastAPI server that shells out to yt-dlp for maximum compatibility. Switch to serverless Cobalt mode for zero-install quick downloads — no Python needed on the client.

Python FastAPI yt-dlp 2024 Cobalt API WebSocket feed
Try Live App
run_gui.py 
# Start the LENLU DLP backend
import subprocess, webbrowser

def main():
    url = "http://127.0.0.1:8000"
    webbrowser.open(url)

    # Spin up FastAPI server
    subprocess.run([
        "uvicorn", "main:app",
        "--host", "0.0.0.0",
        "--port", "8000"
    ])

if __name__ == "__main__":
    main()

# Windows → py run_gui.py
# macOS/Linux → python3 run_gui.py

Dual-mode. One interface.

Local or serverless — the same beautiful GUI works for both.

Local Server Mode

Full power via Python backend

Run the FastAPI server locally. Full yt-dlp support: merging streams, audio conversion, subtitle embedding, custom flags.

Quick Start

Up and running in
under 60 seconds.

Clone the repo, install dependencies, and launch. The script handles the rest — server start, browser open, everything.

PowerShell / cmd 
$git clone https://github.com/lenluarun/lenlu-dlp.git
$cd lenlu-dlp
$pip install -r requirements.txt
$py run_gui.py
# Browser opens automatically
Terminal (zsh) 
%git clone https://github.com/lenluarun/lenlu-dlp.git
%cd lenlu-dlp
%pip3 install -r requirements.txt
%python3 run_gui.py
# Browser opens automatically
Bash 
$git clone https://github.com/lenluarun/lenlu-dlp.git
$cd lenlu-dlp
$pip3 install -r requirements.txt
$python3 run_gui.py
# Browser opens automatically
// Prerequisites

Requires Python 3.9+, pip, and ffmpeg on your PATH for stream merging and audio conversion. yt-dlp is installed automatically via pip.

Auto-launch browser

run_gui.py starts the server and opens the app automatically — one command, fully ready.

No server? Use Serverless

Switch to Serverless mode in the app header. No Python install needed — Cobalt API handles everything.

Or download the ZIP

Direct ZIP download — no git required. Unzip and run.

Static-hostable UI

Deploy local_host.html to Vercel, Netlify, or GitHub Pages — point it at your local backend via the server URL field.

CLI Reference

Console commands.

Master the built-in terminal emulator with these commands. Hit ? in-app for the full shortcut overlay.

CommandDescription
> statusShow current connection mode (Local/Serverless) and active process details.
> clearPurge the output feed buffer in the console tab.
> librarySwitch to and refresh the local downloads media catalog.
> select-folderOpen the native OS file picker to re-route your download directory.
> site-folderRevert download directory to the default project installation path.
> fetchFetch video metadata and available streams for the current URL.
> downloadStart the quick download using the selected preset format.
> batchOpen the Batch Download tab for queuing multiple URLs.
> queueView and manage the active download queue.
> statsOpen the Stats Dashboard with session download metrics.
> settingsOpen the settings tab directly from the console.
> matrixOpen the format matrix tab to inspect all available streams.
> help formatsDisplay the interactive stream format guide and matrix filter tips.
> help downloadShow quick-download usage tips and preset format guide.
> --helpDisplay the full command list with descriptions. Alias: ? key.
